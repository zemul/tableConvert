# tableConvert
Convert HTML Table to Markdown

### usage
```
go install github.com/zemul/tableConvert@latest

$ tableConvert                   
Paste your HTML Table data  here
<table>
    <thead>
<tr>
    <th>name</th>
    <th>age</th>
</tr>
</thead>
<tbody>
    <tr>
        <td>tom</td>
        <td>18</td>
    </tr>
    </tbody>
</table>

| name | age |
| tom | 18 |

ğŸº write clipboard success
```
